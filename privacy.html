Privacy Policy

OpenClassBeta is a desktop application that integrates with Google Classroom.

Data Collection

We only access data necessary to display courses, assignments, and attachments.

Authentication

Authentication is handled via Google OAuth. Tokens are stored securely.

Data Usage

Data is used only within the application to provide functionality.

Third-party services

Some features may use AI APIs (such as OpenRouter) to process content. No personal data is stored permanently.

User Control

Users can revoke access at any time via their Google account settings.
